Battery Indicator Element
-----------------------

Battery status indicator custom element.

- Relies on [`LitElement`](https://lit-element.polymer-project.org/), a sugar class that extends `HTMLElement`.

### Usage
1. `npm ci`
2. `npm start`
3. <http://localhost:8080>
